# Инструкция по запуску локального сервера

Для корректной демонстрации многостраничного сайта с видеофоном рекомендуется запустить локальный веб-сервер.

## Python (версия 3.x)

Откройте терминал в директории проекта и выполните:

```bash
python -m http.server 8000
```

Затем откройте браузер и перейдите по адресу: http://localhost:8000

## Node.js (если установлен)

Установите пакет `http-server`:

```bash
npm install -g http-server
```

Затем выполните:

```bash
http-server
```

Сервер запустится по адресу: http://localhost:8080

## Альтернативные варианты

Если ни один из вышеуказанных способов не работает, можно открыть файл `index.html` напрямую в браузере, но в этом случае видеофон может не отображаться из-за ограничений безопасности браузера.

## Структура сайта

- `index.html` - Главная страница
- `gallery.html` - Страница галереи
- `about.html` - Страница "Обо мне"
