# Lorem Ipsum Dolor Sit Amet

Это многостраничное веб-портфолио (3 страницы) создано с использованием чистого HTML5, CSS3 и JavaScript без использования фреймворков. Проект демонстрирует современные веб-технологии и дизайн.

## Особенности дизайна

- **Первый экран (Hero)**: Содержит фоновое видео с темным оверлеем (rgba(0,0,0,0.6)) для лучшей читаемости текста
- **Glassmorphism**: Эффект матового стекла применен к блокам с описанием работ
- **CSS Grid**: Секция галереи реализована с использованием CSS Grid для отображения карточек рендеров
- **Адаптивность**: Полностью адаптивный дизайн для корректного отображения на мобильных устройствах
- **Методология БЭМ**: Все классы именуются согласно методологии БЭМ
- **Многостраничная структура**: Главная страница, галерея и страница "Обо мне"
- **Футуристичный дизайн**: Использование цветов #9EFFFBFF и #00C9FFFF для создания футуристичного эффекта
- **Lorem ipsum**: Все содержимое представлено в виде стандартного заполнителя

## Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (для навигации между страницами)

## Запуск проекта

Для локального запуска проекта достаточно открыть файл `index.html` в любом современном браузере. Для демонстрации с локальным сервером можно использовать встроенную в Python команду:

```bash
python -m http.server 8000
```

Затем перейти по адресу `http://localhost:8000`

## Адаптивность

Сайт полностью адаптирован для различных размеров экранов:
- Desktop: Полноценное отображение всех элементов
- Tablet: Корректировка сетки галереи и размеров элементов
- Mobile: Одноколоночный макет с оптимизированными размерами

## Цветовая палитра

- Основной цвет: #4a00e0 (темно-фиолетовый)
- Вторичный цвет: #8e2de2 (светло-фиолетовый)
- Акцентный цвет: #00c9ff (голубой)
- Фоновый градиент: от #0f0c29 до #24243e
- Glassmorphism: rgba(255, 255, 255, 0.1) с границами rgba(255, 255, 255, 0.2)
